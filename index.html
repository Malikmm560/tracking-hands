✋ Hand Tracking Project

Using Python, OpenCV, and Mediapipe

Welcome to the Hand Tracking project! This tool is designed to help you experiment with real-time hand tracking, gesture recognition, and creative applications for VR/AR.

🤔 What You Can Do

  • Track Hands in Real-Time: Detect finger joints and wrist movement in real time.
  • Gesture Interactions: Program gestures like thumbs-up 👍 or peace sign ✌️ to trigger events.
  • Explore VR/AR Applications: Use hand tracking for immersive experiences in VR/AR.
  • Sign Language Recognition: Adapt this tool for real-time sign language interpretation.

🚀 Installation Guide

  1. Clone the repository: 
    git clone https://github.com/Malikmm560/tracking-hands.git
  2. Install dependencies: 
    pip install opencv-python mediapipe
  3. Run the script: 
    python hand_tracking.py
